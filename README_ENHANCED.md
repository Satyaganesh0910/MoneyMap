## 🚀 How to Run

1. **Install Dependencies:**
   ```bash
   pip install streamlit streamlit-option-menu matplotlib
   ```

2. **Run the Application:**
   ```bash
   streamlit run app.py
   ```

3. **Open in Browser:**
   The app will automatically open at `http://localhost:8501`
